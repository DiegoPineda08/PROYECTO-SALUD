# PROYECTO DE ANÁLISIS DE SALUD

Este proyecto analiza el dataset **“Lifestyle and Health Risk Prediction”**, el cual contiene información relacionada con estilo de vida, condiciones de salud y factores de riesgo.  
El objetivo del proyecto es explorar los datos y extraer conclusiones significativas.

---

##  Estructura del proyecto

- **data/**  
  Contiene el archivo CSV original con los datos usados en el análisis.

- **notebooks/**  
  Contendrá el análisis exploratorio y visualizaciones en formato Jupyter Notebook.
- **README.md** 
  Documento de presentacion proyecto 
- **requirements.txt**  
  Lista de dependencias necesarias para ejecutar el proyecto.
---

##  Diccionario de la base de datos

| Columna | Descripción |
|--------|-------------|
| Age | Edad del individuo |
| Gender | Género (Male/Female) |
| BMI | Índice de masa corporal |
| Sleep_Duration | Horas promedio de sueño |
| Physical_Activity_Level | Nivel de actividad física |
| Smoking_History | Historial de tabaquismo |
| Alcohol_Consumption | Consumo de alcohol |
| Diet_Quality | Calidad de la dieta |
| Stress_Level | Nivel reportado de estrés |
| Blood_Pressure | Presión arterial |
| Cholesterol_Level | Nivel de colesterol |
| Disease | Presencia de enfermedad (Sí/No) |

 Conclusiones 

* El análisis nos permitió identificar los factores que más influyen en el estado de salud de los pacientes. Entre ellos se destacan la edad, el estilo de vida, los hábitos alimenticios y el nivel de actividad física, que demostraron tener la mayor correlación con los indicadores de bienestar.

* Se evidenció que los hábitos modificables, como el ejercicio y la dieta, tienen un impacto significativo en la mejora de los resultados de salud. Esto sugiere que invertir en educación y programas de promoción puede generar beneficios reales en la población.

* De forma general, los altos niveles de riesgo en la salud se producian por malos habitos en las personas hechos diariamente como lo eran: Fumar, consmumo de azucar, poca realizacion de ejercicio entendiendo que lo que se realiza dia a dia es influyente en el bienestar

* El analisis de esta base de datos permite concluir que aparte de los habitos que se realizan a diario es importante tener en cuenta el tipo de trabajo en el cual estan las personas y donde se presentan mayor riesgo de salud 

* Finalmente, es importante resaltar que mantener hábitos saludables es fundamental para conservar un buen estado de salud. Sin olvidar que también resulta esencial conservar un equilibrio adecuado, especialmente cuando el cuerpo debe enfrentar cambios en la rutina o esfuerzos adicionales. Mantener este balance contribuye a un bienestar constante y sostenible.
