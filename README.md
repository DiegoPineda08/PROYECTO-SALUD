# PROYECTO DE ANÁLISIS DE SALUD

Este proyecto analiza el dataset *Lifestyle and Health Risk Prediction* basado en información de estilo de vida y factores de riesgo.

## Estructura del proyecto
- **data/**: contiene el archivo CSV con los datos originales.
- **notebooks/**: aquí estará el análisis en Jupyter Notebook.
- **requirements.txt**: librerías necesarias para ejecutar el análisis.
- **README.md**: documentación general del proyecto.

## Cómo ejecutar el notebook

### 1. Instalar dependencias:
pip install -r requirements.txt

### 2. Abrir el notebook:
jupyter notebook
